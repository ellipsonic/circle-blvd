circle blvd.
========

### Prerequisites
1. Node 0.8
2. CouchDB

### Installation
* `git clone https://github.com/secret-project/circle-blvd.git`
* `cd circle-blvd/app/`
* `npm install`
* `(start CouchDB)`
* `node server.js`
* `Go to http://localhost:3000`

### Testing
* Uses `karma` and `jasmine` for now.
* `scripts/test.sh`
